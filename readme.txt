How to use:

- Download the whole repository.
- Go to your user Maniaplanet folder (somewhere like "Documents/Maniaplanet").
- Then, go to the "Skins" folder.
- Unzip the "Any" and "stadium" folders from the downloaded repository into the "Skins" folder of your user Maniaplanet folder.
- Restart Maniaplanet.

Ta-daa
